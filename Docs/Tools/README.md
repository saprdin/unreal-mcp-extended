# Unreal MCP Tools

This document is an index to all the tools supported. Documentation pages mirror the Python tool modules in `Python/tools/`.

- [Editor Tools](editor_tools.md) - Actors in the level, assets, Play-In-Editor control, and debugging (output log, console commands, editor Python)
- [Blueprint Tools](blueprint_tools.md) - Blueprint classes, components, properties, physics, and materials
- [Node Tools](node_tools.md) - Blueprint event-graph editing: nodes, variables, connections, and introspection
- [UMG Tools](umg_tools.md) - Widget Blueprints for game UI: text, buttons, images, progress bars, and event bindings
- [Project Tools](project_tools.md) - Project-wide settings such as input mappings
